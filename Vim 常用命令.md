### 宏命令(Macros)

高级一些的编辑器，都会包含宏功能，vim 当然也不能缺少了，在 vim 中使用宏是非常方便的：

| 命令 | 说明                             |
| ---- | -------------------------------- |
| :qx  | 开始记录宏，并将结果存入寄存器 x |
| q    | 退出记录模式                     |
| @x   | b播放记录在 x 寄存器中的宏命令   |

当在 `normal` 模式下输入 `:qx` 后，你对文本的所有编辑动作将会被记录下来，再次输入 `q` 即退出记录模式，然后输入 `@x` 对刚才记录下来的命令进行重复，此命令后跟数字，表示要重复多少次，比如 `@20` ，可以重复 20 次。这个在文本的批处理中是非常有用的。
